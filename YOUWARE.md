# Mindful Knowledge Studio - 项目说明

## 项目概览
- **项目类型**：个人知识管理 Web 主页与分类笔记体验（React + TypeScript + Tailwind CSS）
- **主要受众**：需要安静、结构化笔记空间的个体学习者与创作者
- **核心目标**：以柔和的色彩与插画呈现个人知识体系，突出灵感捕捉、知识链接、自我复盘与分类浏览体验，并串联宁静的笔记模板

## 已实现的界面要点
1. **沉浸式英雄区**
   - 渐变背景与柔焦色块营造静谧氛围
   - 展示插画、今日笔记提示卡、动态统计指标与分类快捷入口
   - 主按钮「开始整理」直达新建笔记模板，副按钮跳转至分类总览
2. **功能导览（Features）**
   - 三列卡片说明灵感捕捉、语义链接、复盘仪表板
   - 每张卡片配有 Emoji 图标与浅色强调底
3. **知识温室（Library）**
   - 分类卡片改为全局导航，悬浮样式与动态笔记数量对齐
   - 侧边提示卡提醒自动复盘计划，并引导浏览更多分类
4. **社群触角**
   - 三个社交媒体链接卡片，覆盖 X、Instagram、YouTube
   - 卡片含平台徽章、频道名称与内容说明，全部新窗口打开
5. **多页面结构与导航**
   - 新增粘性顶栏，串联主页、分类馆与新建模板，交互统一悬浮视觉
   - Notes Library 页面按分类展示笔记卡片与示例内容，支持返回主页与快速新建
   - 分类详情页列出对应主题的所有笔记，展示心情标签、文本摘要与时间戳
6. **宁静笔记模板**
   - 表单提供标题、分类、正文、标签、心情字段
   - 提交后写入内存存储并跳转至对应分类页
   - 输入框与背景沿用柔和米白与墨绿配色
7. **全局数据支撑**
   - 内置 notes API（Zod 校验 + 内存存储）提供分类、笔记创建与读取
   - 首次载入自动注入示例笔记，确保分类页面展示完整
8. **页脚**
   - 版权信息与品牌宣言，维持页面宁静语调

## 视觉与交互基调
- **色彩**：主色 #5E7D7E（墨绿）、辅助暖米色 #F4F0EB、冷色高光 #BFD8D5，搭配白色与深灰中和
- **排版**：中文标题使用中粗字体；正文维持 14–18px 的舒适行距
- **布局**：12 栏响应式网格；主页与分类页卡片均采用悬浮卡式设计
- **动效**：按钮与卡片在悬停时轻微上抬并增强阴影，强化可交互提示
